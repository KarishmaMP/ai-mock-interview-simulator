# 📘 AI Mock Interview Simulator

A team-based web application developed to help students prepare for technical and HR interviews through interactive assessment modules.

---

## 👥 Team Project
Developed collaboratively as part of academic coursework, with contributions across multiple modules including UI design, core logic, and testing.

---

## 🚀 Project Description
The AI Mock Interview Simulator is a web-based platform designed to simulate real interview scenarios.  
It features MCQ-based technical assessments with randomized questions, along with a behavioral interview module that uses speech recognition to analyze user responses and provide feedback.
